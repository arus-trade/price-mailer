# 📧 Сервис автоматической рассылки прайс-листов

[![Статус сборки](https://github.com/YOUR_USERNAME/YOUR_REPOSITORY/actions/workflows/mailer.yml/badge.svg)](https://github.com/YOUR_USERNAME/YOUR_REPOSITORY/actions/workflows/mailer.yml)

Автоматический сервис, который находит на почте письмо с прайс-листом и пересылает его заданным получателям. Работает полностью автономно благодаря **GitHub Actions**.

---

## ✨ Как это работает

Скрипт выполняет простую последовательность действий:
1.  **Подключается** к почтовому ящику по защищенному протоколу IMAP (Mail.ru).
2.  **Находит** последнее письмо с заданной темой (например, «Прайс-лист»).
3.  **Скачивает** прикрепленный к письму Excel-файл (`.xls`, `.xlsx` и др.).
4.  **Пересылает** этот файл получателям из списка, используя почтовый ящик Яндекс (SMTP).

Процесс запускается автоматически по расписанию или может быть запущен вручную в любой момент.

---

## 🚀 Установка и запуск (метод GitHub Actions)

Это основной и рекомендуемый способ запуска. Он не требует собственного сервера, работает в облаке GitHub и надежно защищает ваши пароли.

### Шаг 1: Подготовка репозитория

Если вы разворачиваете проект с нуля, сделайте `fork` этого репозитория или используйте его как шаблон.

### Шаг 2: Настройка секретов

Это самый важный шаг. Вам нужно сообщить GitHub Actions ваши логины и пароли, не записывая их в код. Для этого используются **секреты (Secrets)**.

1.  Перейдите в настройки вашего репозитория: `Settings` > `Secrets and variables` > `Actions`.
2.  Нажмите кнопку **`New repository secret`** и последовательно создайте следующие секреты:

| Имя секрета       | Описание                                                         | Пример значения                               |
|-------------------|------------------------------------------------------------------|-----------------------------------------------|
| `IMAP_LOGIN`      | Логин ящика **Mail.ru**, куда приходят прайс-листы.              | `price@example.com`                           |
| `IMAP_PASSWORD`   | **Пароль приложения** для IMAP-ящика (Mail.ru).                  | `ab12cd34ef56gh78`                            |
| `SMTP_LOGIN`      | Логин ящика **Яндекс**, с которого идёт отправка.                | `sender@yandex.ru`                            |
| `SMTP_PASSWORD`   | **Пароль приложения** для SMTP-ящика (Яндекс).                   | `xy98zw76vu54ts32`                            |
| `FROM_ADDRESS`    | Email-адрес отправителя в заголовке письма (обычно = SMTP_LOGIN).| `sender@yandex.ru`                            |
| `RECIPIENTS`      | Список email-адресов получателей в формате JSON-массива.         | `["user1@example.com", "user2@example.com"]`  |

> **⚠️ Важно:**
> *   Используйте **пароли приложений**, а не основные пароли от почтовых ящиков. Инструкции для Mail.ru и Яндекс легко найти в их справочных центрах.
> *   Значение для `RECIPIENTS` должно быть строкой в формате JSON-массива, как в примере.
> *   `FROM_ADDRESS` — это адрес, который увидит получатель в поле «От кого». Обычно совпадает с `SMTP_LOGIN`.

### Шаг 3: Запуск!

Готово! Теперь сервис будет работать автоматически.

*   **Автоматический запуск:** Происходит по расписанию, заданному в файле `.github/workflows/mailer.yml` (по умолчанию — каждый день в ~05:55 МСК).
*   **Ручной запуск:**
    1. Перейдите на вкладку `Actions` в вашем репозитории.
    2. В меню слева выберите воркфлоу `Price Mailer Cron`.
    3. Нажмите кнопку `Run workflow` и подтвердите запуск.

Вы можете следить за ходом выполнения и логами на вкладке `Actions`.

## 💻 Локальный запуск (для отладки)

Если вы хотите протестировать скрипт на своем компьютере, выполните следующие шаги:

1.  **Склонируйте репозиторий:**
    ```bash
    git clone https://github.com/YOUR_USERNAME/YOUR_REPOSITORY.git
    cd YOUR_REPOSITORY
    ```

2.  **Создайте файл конфигурации:**
    В папке `price_mailer/run` создайте файл `config.json` по этому шаблону. **Не добавляйте этот файл в коммиты (`git add`)!**

    ```json
    {
        "imap": {
            "server": "imap.mail.ru",
            "port": 993,
            "login": "ВАШ_IMAP_ЛОГИН",
            "password": "ВАШ_IMAP_ПАРОЛЬ_ПРИЛОЖЕНИЯ"
        },
        "smtp": {
            "server": "smtp.yandex.ru",
            "port": 465,
            "login": "ВАШ_SMTP_ЛОГИН_ЯНДЕКС",
            "password": "ВАШ_SMTP_ПАРОЛЬ_ПРИЛОЖЕНИЯ",
            "from_address": "ВАШ_SMTP_ЛОГИН_ЯНДЕКС"
        },
        "search_subject": "Прайс-лист",
        "recipients": [
            "user1@example.com",
            "user2@example.com"
        ],
        "email_subject": "Актуальный прайс-лист",
        "email_body": "Добрый день!\n\nВо вложении актуальный прайс-лист.\n\nС уважением."
    }
    ```

3.  **Запустите скрипт:**
    Убедитесь, что у вас установлен Python 3.8+.

    ```bash
    # Находясь в корневой папке репозитория
    python price_mailer/run/price_mailer.py
    ```
    Скрипт создаст лог-файл в папке `price_mailer/run/logs/`.
